## 표현식
표현식이란 값으로 평가될 수 있는 문입니다. 즉, 새로운 값을 생성하거나 값을 참조합니다.

```javascript
var score = 100; // 100은 그 자체로 표현식이다. (리터럴)

score; // 100이라는 값으로 평가되므로 표현식이다.
```

값으로 평가될 수 있는 문은 모두 표현식입니다.

```javascript
// 리터럴 표현식
10
'hello'

// 식별자 표현식( 이미 선언되었다고 가정 )
score;
person.name;
arr[1]

// 연산자 표현식
10 + 10;
sum = 10;
sum !== 10;

// 함수/메소드 호출 표현식
square();
person.getName();
```

## 문
문은 프로그램을 구성하는 기본 단위이자 **최소 실행 단위**입니다.
문은 여러 토큰으로 구성됩니다. 토큰은 문법적인 의미를 가지며, 문법적으로 더이상 나눌 수 없는 코드의 기본요소입니다.

```javascript
// 변수 선언문;
var x ; //var 과 x 가 토큰에 해당된다.

// 할당문
x = 5;

// 조건문
if( x > 1 ){
  console.log(x);
}

// 함수 선언문
function foo(){}
```
표현식은 문의 부분집합이라고 할 수 있죠.

## 표현식인 문과 아닌 문

구별하는 법은 변수에 할당해 보면 됩니다.

```javascript
// 변수 선언문은 표현식이 아닌 문이다.
var x;
var exp = var x; // error

// 할당문은 표현식인 문이다.
x = 5;
var exp = x = 5; // 가능
console.log(exp); // 5
```
